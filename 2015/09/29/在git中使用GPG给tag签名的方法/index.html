在git中使用GPG给tag签名的方法

平台：windows
git：官版 git
gui：smartgit
GPG：gungpg

gungpg默认key存储位置在c:\Users\<用户名>\AppData\Roaming\gnupg\

而git默认访问位置在c:\Users\<用户名>\.gnupg\

所以需要使用mklink创建链接从实际位置指向git访问位置

git默认使用的签名是与用户名和邮箱对应的签名
